# excel-ent

## Installation of the project :

```sh
cd backend && npm install
cd ../frontend && npm install
```

## Files to add

-   backend/.env :

```sh
SRV_PORT = 8888

DB_NAME = 'excel'
DB_USER = 'excel_user'
DB_PASS = 'excel_pass'
DB_HOST = 'localhost'

BCRYPT_SALT_ROUNDS = 12

JWT_SECRET = "secret"
JWT_DURATION = "3600"
```
